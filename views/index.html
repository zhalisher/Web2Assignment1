Our website is an movie platform where users can watch films and TV series, leave reviews, and create personal lists of favorite movies. The platform provides detailed information about each film.

Feature Info
Register/Log in Allows users to create a new account or sign in to an existing one in order to access personalized features.
Watch Movie Enables users view available movies and TV series directly on the platform.
Description of Movie/TV series Provides detailed information about each film, including plot description, cast, genre, release year, and other relevant details.
Reviews Displays user or critic reviews for each movie and allows users by API.
Mark Films Lets users bookmark, like, or mark films so they are saved in a different list.
